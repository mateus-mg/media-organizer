# CLI Manager (`cli/cli_manager.py`)

Interactive menu system for Media Organizer.

## Main Menu Options

| Option | Description |
|--------|-------------|
| `[1]` | Organize media files |
| `[2]` | Filename suggestions |
| `[3]` | System information (stats, logs, quality) |
| `[4]` | Genre catalog management |
| `[5]` | Playlists (Navidrome) |
| `[6]` | Exit |
| `[9]` | Admin: Music genre backfill (if enabled) |

## Sub-Menus

### Genre Catalog Management

- View/edit invalid genres blacklist
- View/edit musical keywords catalog
- View/edit genre exceptions
- Run genre quality reports

### System Information

- Statistics dashboard
- Log viewer
- Quality monitor
- Genre quality report
