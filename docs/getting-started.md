# Getting Started

Welcome to Media Organizer! This guide will help you get up and running.

## Prerequisites

- Python 3.9+
- Linux (hardlink support required)
- 4GB RAM minimum
- Media files to organize

## Quick Start

1. [Install](guides/installation.md) the application
2. [Configure](configuration/index.md) your paths
3. Run your first organization: `./run.sh organize`

## What Media Organizer Does

Media Organizer automatically:
- Scans download directories for new media
- Classifies files by type (music, books, comics)
- Validates file completeness and quality
- Enriches metadata from online sources
- Organizes files into structured libraries
- Creates hardlinks to save disk space
- Syncs playlists to Navidrome
